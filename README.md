Welcome to matrix rush. Download the files and run the file "main.exe" to start the game.

This is essentially a race against an AI. You are to complete the given matrix question in the time alotted.
If you answer correctly, your player character will move up one slot.

If you however:

- Run out of time.
- Answer incorrectly.
Your player square will stay stagnant, but the AI opponent will move up.

This game is intended to help you complete matrix equations in a fun manner against the AI. Make sure you are checking your work before proceeding.
